## Sections

1. **Hero Section** - Introduction and call-to-action buttons
2. **About** - Personal information and statistics
3. **Projects** - Showcase of featured projects
4. **Skills** - Technical skills and technologies
5. **Contact** - Contact form and social links

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content to match your information

## Customization

### Personal Information

1. **Name and Title**: Update in the hero section (`index.html`)
   - Change "Your Name" to your actual name
   - Update the subtitle to your role/title

2. **About Section**: Modify the about text and statistics
   - Update the description paragraphs
   - Change the stat numbers and labels

3. **Projects**: Replace the project cards with your own projects
   - Update project titles, descriptions, and tags
   - Add links to your live demos and GitHub repositories
   - Replace placeholder images with actual project screenshots

4. **Skills**: Update the skills and technologies
   - Modify the skill categories and items
   - Add or remove skills as needed

5. **Contact Information**: Update contact details
   - Change email, phone, and location
   - Update social media links (GitHub, LinkedIn, Twitter)

### Styling

- **Colors**: Modify CSS variables in `styles.css` (root section)
  - `--primary-color`: Main brand color
  - `--secondary-color`: Secondary accent color
  - `--text-primary`: Main text color
  - `--text-secondary`: Secondary text color

- **Fonts**: The site uses Google Fonts (Inter). Change the font import in `index.html` if desired.

### Images

Replace the SVG placeholders with actual images:
- Hero section: Add a professional photo or illustration
- Project cards: Add screenshots of your projects
